# Python Web Scraping Examples

This repository contains simple examples of Python scripts used for:

• Web scraping from websites
• API data extraction
• Browser automation

## Projects

### 1. Ecommerce Scraper
Extracts product titles and prices from a sample ecommerce website.

### 2. API Data Extractor
Collects structured data from a public REST API.

### 3. Browser Automation Bot
Uses Selenium to automate browser interaction and collect page data.

## Technologies

Python  
Requests  
BeautifulSoup  
Pandas  
Selenium
